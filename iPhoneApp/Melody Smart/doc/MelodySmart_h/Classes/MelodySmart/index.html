sendData:

- (bool) sendData:(NSData*) data;
Parameters
data

The data to send to the peripheral. Due to BLE protocol limitations, the maximum size of a single data chunk is 20 bytes. No data is transmitted if the chunk exceeds that size.

Discussion

Sends data using the Data service to the connected Melody peripheral.
